SCC: Sistema para Controle de Covid


Contexto central do projeto


  O conceito do projeto surgiu como exigencia da faculdade, o projeto foi criado como uma forma de avaliação conechecida como projeto integrado multidisciplinar (PIM) para faculdade, o PIM envolve todas materias envolvidas no bimestre no bimestre em questão, estava sendo estudado programação em C e metodologias ageis.


Problema a ser resolvido


  Por conta da recente pandemia causada pela Covid-19, houve uma grande procura de métodos para diminuir e controlar a contaminação, com isso a área da saúde e tecnologia ficaram lado a lado para encontrar formas de controlar o número de casos, seguindo esse contexto o programa aqui apresentado foi desenvolvido.

  Seguindo os parâmetros estabelecidos, o programa foi desenvolvido na linguagem C, uma linguagem antiga mas funcional nos dias de hoje principalmente por ser considerada uma linguagem de baixo nível, o que acaba permitindo acesso mais fácil a recursos desse nível, como o acesso à memória da máquina.

  O programa foi desenvolvido para instituições na área da saúde, com a finalidade de cadastrar pacientes infectados pela covid-19. Ao iniciar o programa ele deve pedir login e senha em seguida ele deve realizar o cadastro, assim o usuário deve informar as seguintes informações sobre o paciente:

● Nome;
● CPF;
● Telefone:
● Endereço (rua, número, cidade, estado e CEP);
● Data de nascimento;
● Data de diagnóstico.

  Caso o paciente possua alguma comorbidade e se possua idade maior ou igual a sessenta e cinco anos as seguintes informações deverão ser armazenadas em um arquivo para ser enviado a secretaria da saúde da cidade, nome, idade e CEP.


Principais funções: Validação de login


A função criada permite ou nega o acesso, mantendo sempre o usuário padrão como “Unip” e senha “Epsilon”. Uma possível melhoria seria a criação de um banco de dados para armazenar novos acessos para usuário.


Principais funções: Armazenamento de dados


  Os dados digitados pelo usuário em um arquivo texto algo funcional, mas como melhoria ao invés de armazenar as informações em um arquivo texto poderia ser armazenado em uma tabela, assim as informações serão armazenados diretamente na tabela, podendo ser acessada diretamente na mesma, não sendo mais necessário o uso de matriz.
